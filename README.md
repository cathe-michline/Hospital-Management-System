# Hospital-Management-System
This project is a management system for hospitals so that users can easily view the hospitals and make appointments to a specific doctor present in that particular hospital in a selected time. 

**PROJECT MODEL**
PATIENT:
1.	Login/signup
2.	Homepage 
  a.	Hospitals
3.	Hospital Page
  a.	Make appointment
4.	Profile
  a.	Booked Appointments

DOCTOR:
1.	Login
2.	Profile page
  a.	Today’s appointments
  b.	Upcoming appointments
  c.	Patient’s history
3.	Patients History 
  a.	Previous visits of patients (if any)
  b.	Next appointments scheduled (if any)

HOSPITAL:
1.	Login
2.	Homepage  
  a.	Available doctors
  b.	Add Doctors to the hospital
  c.	That days appointments
